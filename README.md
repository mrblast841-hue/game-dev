# Arena Survivor Game

A 2D survival game developed using the Godot Engine where players must survive against waves of enemies for as long as possible. The project was created to explore game development concepts such as player movement, enemy AI, collision detection, health systems, and gameplay balancing.

## Features

* Smooth player movement
* Enemy spawning system
* Health and damage mechanics
* Collision detection
* Increasing gameplay challenge over time
* Simple and intuitive controls
* User interface elements for gameplay feedback

## Technologies Used

* Godot Engine
* GDScript

## Gameplay

The player is placed in an arena where enemies continuously spawn and attempt to attack. The objective is to survive for as long as possible while avoiding enemies and managing health effectively.

## Controls

| Action     | Key |
| ---------- | --- |
| Move Up    | W   |
| Move Down  | S   |
| Move Left  | A   |
| Move Right | D   |

## Project Structure

* `Player/` – Player movement and health scripts
* `Enemy/` – Enemy behavior and attack logic
* `Spawner/` – Enemy spawning system
* `UI/` – Health bars and interface elements
* `Scenes/` – Main game scenes
